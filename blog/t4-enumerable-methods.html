The Cycle Enumerable

What it is and how to use it

July 12, 2015

The Enumerable cycle is a public method that calls a block for each element repeatedly a select number of times or forever if none or nil is given. If a non-positive number is given or the collection is empty, this does nothing. The Enumerable cycle also returns nil if the loop has finished without getting interrupted. If no block is given, an enumerator is returned instead.

So in this first example of cycle, since there is no select number given, the method will print out the array a forever. In the second example, it is given a number of 2 times so it will print out the array twice:



a = ["a", "b", "c"]

a.cycle {|x| puts x }
#prints: a, b, c, a, b, c,.. forever.

a.cycle(2) {|x| puts x }
#prints: a, b, c, a, b, c.
