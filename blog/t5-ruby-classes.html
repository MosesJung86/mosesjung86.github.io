Creating and Calling upon a Class

July 18, 2015

This is an example of creating your own class, along with its instance variables and instance methods. This class takes an input of an array and outputs a random generated element from that array (think of it as rolling a dice with words and/or symbols as its sides):

 

class Dice

#This is how to create a class named "Dice"

   def initialize (input)

#First we need to initialize the class by setting up input variables

     @input = input

     @count = input.count - 1

#These are instance variables, they are set inside initialize, cannot be modified but can be used anywhere in this class. To use this instance variable in any of the future methods, we would use the '@' symbol followed by the variable name.

     if sides < 1

       raise ArgumentError.new("Number cannot be less than 1")

#This is to create an Argument Error if the input is less than 1, because you cannot have a dice with less than 1 side

     end

   end

   def sides

     return @count + 1

   end

#This is an instance method, it is defined inside the class and can be called upon only through the class. In this method, it is returning how many sides the dice has.

   def roll

     num = Random.new

     index = num.rand(0..@count)

     return @input.fetch(index.to_i)

#This is also another instance method, and this method produces a random generated output of an element from that array input. Notice how we use '@input' to call upon the instance variable set for the input.

     end

   end

end

array = ["Hi",2,"stuff",4,true,"things",6,false]

#Now to call upon the class we need to create an object of that class like so:

d = Dice.new(array)

#Then we can call on any method inside the class like this:

puts d.sides

puts d.roll



Creating your own class is useful in many ways. Instead of having to call a variety of methods constantly, you can just create a class with a set of methods and just call upon the class methods. If done correctly, it should save time and coding space. It is also useful if you end up having many different classes so that each method inside the classes may have the same name but may work differently.

This is the class above but without the comments just to show you how it would normally look like:

 

class Dice

   def initialize (input)

     @input = input

     @count = input.count - 1

     if sides < 1

       raise ArgumentError.new("Number cannot be less than 1")

     end

   end

   def sides

     return @count + 1

   end

   def roll

     num = Random.new

     index = num.rand(0..@count)

     return @input.fetch(index.to_i)

     end

   end

end

array = ["Hi",2,"stuff",4,true,"things",6,false]

d = Dice.new(array)

puts d.sides

puts d.roll
