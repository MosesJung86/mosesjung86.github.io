Ruby Basics
Variables:
  • Integers: my_num = 3
  • Boolean: my_boolean = true
  • Strings: my_strings = "Hi"
Math:
  • Addition (+): add = 3 + 4
  • Subtraction (-): subtract = 5 - 2
  • Multiplication (*): multiply = 6 * 3
  • Division (/): divide = 6 / 3
  • Exponentiation (**): exp = 3 ** 2
  • Modulus (%): mod = 6 % 3
  • PEMDAS (operator precedence): Parenthesis, Exponential, Multiplication, Division, Addition, Subtraction
Print, puts & p:
  • print: takes value and prints it to the screen
  • puts: adds a new (blank) line after the value you want it to print. puts generally prints the result of applying to_s on an object.
  • p: returns value and prints out to the screen. p prints the result of inspecting the object.
